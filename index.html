Backend

I specialize in backend development for web applications — from designing architecture to implementing and optimizing business logic. My focus is on reliability, scalability, and clean, maintainable code.

My main tech stack includes C# / .NET, ASP.NET Core, Entity Framework, and MS SQL Server. I also use Blazor and Windows Forms when client-side functionality is needed.

I'm familiar with Docker, REST APIs, JWT authentication, endpoint protection, logging, and database migrations. I prefer clear separation of concerns and a modular architecture.

I enjoy building backend systems that not only work, but are also easy to maintain. I believe a good server is one the user never notices — because it's fast and dependable.
